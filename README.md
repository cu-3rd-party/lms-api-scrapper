# lms-api-scrapper

## В общем, как работать
1. Скачиваем себе, загружаем плагин по пути lms-api-scapper/extension, открываем вкладку с LMS
2. Заходим в меню плагина и нажимаем Start monitor
3. Самостоятельно руками проходим по всем вкладкам LMS. Открываем домашки, файлы, лонгриды, профиль и тд. Если есть возможность открыть две домашки - откройте две, тогда скрипт произведет категоризацию типа GET /api/lms-api/materials/{id}
4. После того, как всё обошли, открываем меню плагина и нажимаем Stop monitor
5. Сохраняем, не меняя названия в папку с gen_api_docs.py, запускаем gen_api_docs.py
6. Потом открываем в любом MD редакторе сгенерированный файл и смотрим API документацию LMS

# NOTES: Тестировалось на Google Chrome
